========================================
INSTALLATION ANGULAR MATERIAL + TAILWIND
========================================

=== ANGULAR MATERIAL ===

# Installation automatique (recommandée)
ng add @angular/material

# OU installation manuelle
npm install @angular/material @angular/cdk @angular/animations

=== TAILWIND CSS ===

# Installation des dépendances
npm install -D tailwindcss postcss autoprefixer

# Initialisation de la configuration
npx tailwindcss init -p

=== CONFIGURATION TAILWIND ===

Créer/Modifier le fichier : tailwind.config.js

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./src/**/*.{html,ts}"],
  corePlugins: {
    preflight: false, // Important pour éviter les conflits avec Material
  },
  theme: {
    extend: {},
  },
  plugins: [],
}

=== STYLES CSS ===

Ajouter au début de src/styles.css :

@tailwind base;
@tailwind components;
@tailwind utilities;

/* Vos styles Angular Material resteront en dessous */

=== BONNES PRATIQUES ===

✅ Utilisez Angular Material pour :
- Composants d'interface (buttons, cards, dialogs)
- Navigation (toolbar, sidenav, tabs)
- Formulaires (input, select, datepicker)

✅ Utilisez Tailwind CSS pour :
- Layout et positionnement (flex, grid)
- Espacement (p-4, m-2, gap-6)
- Couleurs personnalisées
- Responsive design
- États hover/focus
