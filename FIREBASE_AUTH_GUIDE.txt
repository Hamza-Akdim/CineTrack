========================================
GUIDE FIREBASE AUTHENTICATION - CINETRACK
========================================

=== 1. COMMANDES À EXÉCUTER ===

# Générer les services et composants
ng generate service services/auth
ng generate guard guards/auth
ng generate component components/login
ng generate component components/register
ng generate component components/dashboard

# Installer les dépendances supplémentaires si nécessaire
npm install @angular/forms @angular/router

=== 2. CONFIGURATION FIREBASE CONSOLE ===

1. Aller sur https://console.firebase.google.com/
2. Créer un projet "cinetrack-app"
3. Ajouter une application Web
4. Aller dans Authentication > Sign-in method
5. Activer:
   - Email/Password
   - Google (optionnel)
6. Copier la configuration Firebase
